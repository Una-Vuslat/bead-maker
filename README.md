# bead-maker
MARK 拼豆智能生成器 (MARK Pixel Bead Workbench)
一款专为拼豆玩家打造的专业级图纸转换与库存管理工具。通过智能算法解决“色号短缺”痛点，助你用手头现有的豆子拼出最完美的艺术品。

🌟 核心亮点
🎨 MARK 全系列色库：内置完整 291 种 MARK 豆标标准色号数据（A-ZG系列）。

🤖 智能库存替换：根据你勾选的现有库存颜色，自动计算出视觉最接近的替代方案。

👁️ 人类视觉算法：底层采用 CIELAB 色彩空间（而非简单的 RGB）进行颜色匹配，确保生成的图纸色彩自然，拒绝“发脏发灰”。

📐 专业正交坐标系：符合玩家习惯，以图纸左下角为原点 (1,1)，向上、向右递增，方便快速定位。

🔍 无级缩放视窗：支持 10% - 500% 自由缩放及“适应视图”模式，大颗粒图纸也能全局全览。

💾 高清带清单导出：一键生成包含“像素网格图纸”与“所需色号用量清单”的高清 PNG 图片。

🛠️ 功能详解
1. 库存管理系统
本地存储：自动记忆你的库存勾选，关闭浏览器不丢失数据。

数据迁移：支持导出/导入 .json 配置文件，轻松实现跨设备同步。

🚀 快速开始
由于本项目是纯前端开发，无需安装任何环境：

访问项目网址： https://una-vuslat.github.io/bead-maker/

在左侧上传你想要转换的图片。

调整“横向颗粒数”（建议 50-100）。

在下方“色号管理”中，点击 [ ➕ ] 将你拥有的豆子加入“我的库存”。

点击 [ 🚀 一键生成专业拼豆图纸 ]。

💻 技术栈
语言：HTML5, CSS3, JavaScript (ES6+)。

图形：Canvas API（像素级渲染与图像合成）。

算法：CIE76 色差公式 (Delta E) 在 LAB 空间下的应用。
